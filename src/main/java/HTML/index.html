Experimental database

Purpose
Our goal is to provide researchers with a tool to streamline their research, minimise wasteful use of their time and funding, all the while minimising animal testing!

How to use the page
Use the search bar to search for literature (for example from a particular author, or from publications from a specific year!) You can also leave it empty to display all the available papers!
Be sure to select the appropriate cytokine, so we know where to look!

Contact us at cytodb@outlook.com to report bugs and suggest new features!
